# Changelog - Ad Fontes Manager

## [Unreleased] - 安全升级与架构优化

### 🔐 Security
- **修复变量使用错误**: 单词创建时现在正确存储原始 lemma 值（保留大小写）
- **添加管理员身份验证**: `/api/core/config` 端点现在需要 `x-admin-token` 验证
- **限制动态数据库 URL**: 生产环境禁止 `x-db-url` 请求头，防止 SSRF 攻击
- **连接池安全配置**: 添加最大连接数(20)、空闲超时(30s)、连接超时(5s)限制
- **升级 Express 到 5.0**: 修复已知安全漏洞，提升框架安全性

### ⚡ Performance
- **连接池优化**: 配置合理的连接池参数，防止连接耗尽
- **前端构建优化**: 使用 Vite 构建，代码分割优化加载性能

### 🏗️ Infrastructure
- **Node.js 升级**: 要求 Node.js >= 22.0.0，获得长期支持
- **Docker 镜像更新**: 基础镜像升级到 node:22-alpine

### 📝 Recent Features (Previous Releases)
- **单词服务重构**: 模块化为多文件结构，提升可维护性
- **卡片图片生成**: WordPreview 支持生成并下载单词卡片图片
- **严格 YAML 校验**: 添加单词添加接口的严格 YAML 格式验证
- **搜索功能增强**: 添加精确匹配模式和防抖处理
- **服务端分页**: 实现单词列表的服务端分页和详情按需加载
- **智能冲突解决**: v2 版本引入智能冲突检测和解决机制

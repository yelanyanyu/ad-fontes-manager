## [2.0.3] - 2026-06-03

### 新增

- **AI 搜索工具循环**：Searching 阶段支持工具结果回灌给模型继续总结，避免工具调用成功后仍落入无工具 fallback 输出。
- **批量 Anki 重复决策**：批量导入改为先检查重复项，发现重复后用项目弹窗选择覆盖导入全部或仅导入新卡片。
- **Word Editor 格式修复模块**：新增独立的格式修复命令与校验控制器，覆盖 Fill Editor、手动修复和实时校验场景。
- **共享 UI Template**：新增 renderer 级共享 UI 样式模板，统一按钮、chip、面板、弹窗、空状态和表单控件的基础样式。

### 变更

- AI Generate 抽离批量生成面板、单词生成表单、进度面板和待优化弹窗，主抽屉组件退回为状态协调器。
- Word List 抽离表格、批量 Anki 摘要条、列配置和选择状态模块，列表页后续可继续向仪表盘和批量操作界面扩展。
- Queue Bar 和 Generate Drawer 使用更一致的项目样式，减少局部 CSS 和重复按钮逻辑。
- Word List 默认排序改为 Modified Newest，并将排序选项文案改为更明确的修改时间 / 创建时间语义。
- Stage Details 视图会随左侧单词切换同步展示对应单词的 stage 信息，空数据时如实显示空状态。
- 英语和德语预览不再渲染 `cognate_family.instruction`，该字段仍保留在 schema 和 YAML 数据中。

### 修复

- 修复 YAML 格式修复顺序混乱导致双引号标量丢失的问题，保留英语双引号并避免 `*-ter-` 等内容被误判为 YAML alias。
- 修复中文右双引号等非 ASCII 引号导致 YAML 修复结果再次失效的问题。
- 修复重复 YAML section 和 duplicated mapping key 诊断不够直观的问题，增强重复键与阻塞修复的提示。
- 修复 Word Editor 软换行、行号和实际 YAML 行之间的视觉错位，避免用户按错误视觉行修复缩进。
- 修复德语校验仍要求废弃 `dialectal_notes` 和 `observations` 顶层字段的问题，并对照英语流程更新德语检测与修复机制。
- 修复批量导入 Import 直接导入而不是先检查 Duplicate 的问题，避免 ready 卡片被重复导入。

### 文档

- 更新 README 和项目文档，删减臃肿或过期的 mockup、task 和调研文档。
- 新增声明式 stage policy / agent loop ADR，记录 search 是 agent tool 而不是固定 pipeline stage。
- 更新提示词 schema 与创意提示文档，写入英语、德语和 content reviewer 的最新格式要求。
- 更新架构上下文，记录 Queue metrics、Format Fix、Stage Details、WordList 模块化等术语。

### 移除

- 移除过期 mockup、任务计划和德语调研文档，减少 docs 根目录下对 agent 和开发者没有维护价值的内容。

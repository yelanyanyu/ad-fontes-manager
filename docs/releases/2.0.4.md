## [2.0.4] - 2026-06-06

### 新增

- **Run Metrics**：Queue 列表新增任务运行消耗披露，可展开查看总耗时与 token 消耗，并在生成进度中显示各阶段耗时与 token。
- **AI 味机械检测**：审核阶段新增可配置的 AI 味硬标识检测，并把检测结果传入内容审核提示词，帮助定位具体字段中的生硬表达。
- **批量生成快捷输入**：批量输入面板新增单词草稿快捷插入，自动生成 `context`、`notes` 等编辑结构。

### 变更

- 批量生成帮助与输入面板交互重构，提示、来源选择、预览和创建按钮布局更紧凑。
- Queue Run Metrics 展开状态改为内存内维护，支持单行展开、列头批量展开/收起，以及跨 Queue surface 的稳定行为。
- AI 生成表单在生成后清空单词、上下文、备注输入；Improve Text 后清空 Revision notes。
- 内容审核与英文创作提示词更新，减少过窄的“不是……而是……”检测口径，改为识别更通用的否定式 AI 味标识。

### 修复

- 修复 Electron 界面中批量任务状态条横向拉长、不能随容器自适应的问题。
- 修复 Run Metrics 展开后列头收起无效、全量展开后单行无法收回的问题。
- 修复历史/Today 任务打开后进度面板缺少阶段耗时和 token 消耗的问题。
- 修复点击 Queue 记录后 Single / Batch 切换按钮滚出视口、看起来消失的问题。

### 文档

- 更新 Run Metrics 与 Run Metrics Disclosure 术语说明，为后续仪表盘保留结构化指标语义。

## [2.0.5] - 2026-06-08

### 新增

- **词条无损导入导出**：词表支持按当前选中词条导出约定 JSON，并可从同格式 JSON 导入回数据库，便于迁移到其他应用或跨设备备份。
- **导入冲突决策**：导入词条遇到同 lemma / language 数据时，新增冲突审阅弹窗，可逐条选择跳过或覆盖，并复用现有 YAML diff 预览。
- **发布公告来源**：公告默认读取 GitHub latest release note；当 GitHub Release 暂不可达时，在公告面板内显示非阻塞提示，并保留后续 Gitee fallback source 的配置接口。

### 变更

- 批量生成右侧半屏面板改为紧凑模块布局，输入类型切换、帮助说明、快捷加词、解析统计和创建按钮更适合窄面板操作。
- 词表工具栏与表头布局重构：导入/导出使用图标按钮，刷新按钮移动到 Columns 旁边，减少顶部工具区随窗口宽度漂移的问题。
- Settings 中阶段模型分配、网络搜索和审核阈值的子模块标题去重，保留单一清晰标题。
- 数据库目录切换时，如果目标目录没有 `ad_fontes.db`，会创建空数据库并提示用户，避免继续沿用旧数据库路径。

### 移除

- 移除词表中的 Print Selected 按钮及其 console 输出逻辑。

### 文档

- 更新公告来源、导入冲突决策、数据目录、批量输入提示和浮动 tooltip 等领域与设计说明。

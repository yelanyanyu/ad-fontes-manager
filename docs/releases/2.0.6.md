## [2.0.6] - 2026-06-15

### 新增

- **词条结构版本**：新增 Word Schema Version 体系，保存内容记录 `ad_fontes.word_schema_version`，词表与编辑器可识别旧结构词条并提示重新生成。
- **当前结构参考**：YAML Editor 新增当前 schema 参考面板，可在编辑时查看英语/德语当前结构，帮助对照修复。
- **批量删除词条**：词表支持批量删除已选词条；删除数量超过 20 个时需要输入确认文本，避免误删。
- **schema/prompt 同步技能**：新增项目级 `schema-prompt-sync` skill，用于在手动修改 schema 与 prompt 后同步校验逻辑和版本号。

### 变更

- **保存策略更清晰**：新建词条、AI Generate、Workset 保存和导入新词条必须符合当前 Word Schema Version；已有旧词条可无损更新，但不会自动迁移到新结构。
- **导入策略更宽容**：导入会逐条处理当前结构词条；旧结构或未来结构的新词条会被跳过并给出原因，不阻塞同一文件中的其他当前词条。
- **WordPreview 渲染统一**：Word Preview、Job Result Preview 和 Anki `rendered_html` 共用内置词卡 HTML，英语/德语新字段渲染保持一致。
- **YAML Editor 架构重构**：编辑会话、保存意图、校验控制器和编辑器表面拆分为独立模块，为后续切换 CodeMirror 预留结构。
- **WordList 架构整理**：批量删除和 Select All Matching 逻辑抽离为独立模块，降低词表组件复杂度。
- **弹窗层级规则**：AI Generate、schema 参考和公告等浮层按最近点击顺序覆盖，避免相互遮挡。
- **德语结构更新**：德语 schema、prompt 和 WordPreview 样式同步更新，德语词条展示更接近英语卡片风格。

### 修复

- 修复 AI 阶段 YAML 合并失败时退回简单拼接，导致两个顶层 `etymology` 并存的问题。
- 修复重复键诊断把不同 `word_formation.derivations` 列表项里的 `language` 误判为重复字段的问题。
- 修复 YAML Editor 前端解析失败后只显示 `No safe repair`，不显示后端具体 YAML 错误的问题。
- 修复编辑当前词条或新增词条时旧结构标志可能错误显示的问题。
- 修复小屏幕下 YAML Editor 顶部状态和操作按钮拥挤错位的问题。
- 修复界面空白、公告被 AI Generate/schema 参考遮挡等浮层与布局问题。
- 修复 WordPreview 未显示 `word_forms`、对象形 `source_word`、`word_formation` 等新结构字段的问题。

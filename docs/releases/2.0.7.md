## [2.0.7] - 2026-06-25

### 新增

- **CodeMirror YAML Editor**：YAML Editor 迁移到 CodeMirror，提供更稳定的行号、软换行、Tab 缩进和编辑体验。
- **行内错误标记**：可定位的 YAML 语法错误和 Schema 错误会在编辑器中用红色波浪线标出，并保留错误列表用于查看完整说明。
- **编辑器测试路由**：新增 `/test` CodeMirror YAML 测试界面，用于在替换主编辑器前验证输入、缩进、换行和诊断行为。

### 变更

- YAML Editor 不再使用旧的 textarea 行号镜像和软换行测量逻辑，编辑器滚动、选区和只读状态统一由 CodeMirror 管理。
- YAML 层级面包屑继续保留，但旧的缩进深度行号提示被移除，避免与 CodeMirror 自带行号和诊断 gutter 重复。

### 修复

- 修复前端 YAML 解析失败时只更新状态、不生成行内诊断，导致 `nuance:啊` 等语法错误没有红线提示的问题。
- 修复后端返回 `root` 路径的 Schema 诊断无法映射到具体字段的问题；当错误信息以字段路径开头时，编辑器会尝试定位对应 YAML 行。
- 修复点击错误列表时部分可定位诊断无法跳转到 CodeMirror 对应行的问题。

### 文档

- 更新 README 和教程中的 YAML Editor 说明，说明 CodeMirror 行内错误标记行为。

### 移除

- 移除旧 textarea 编辑器实现、textarea 专用 Tab 缩进 helper 和软换行行高测量代码。
